Python Syntax > Conditionals
=============================

Examples
--------

```python
print(True)
print(False)

print(True or False)
print(True and False)

if (45 == 1):
    print("Yes, 45 == 1")
else:
    print("No, 45 != 1")

print("a" == "b")
print(5 > 1)
```
