Loops & Collections
-----

```python
print range(10)

for x in range(10):
    print(x)

sentinel = 4
current = 0
while current < sentinal:
    print(current)
    current += 1
print("done!")
```

Conditionals
------------

```python
print(True)
print(False)

print(True or False)
print(True and False)

if (45 == 1):
    print("Yes, 45 == 1")
else:
    print("No, 45 != 1")

print("a" == "b")
print(5 > 1)
```
